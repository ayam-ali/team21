General Info

Rimplex was designed to work with complex numbers and perform simple operations on them. In order to place a complex number as a single operand in an expression, one must place the complex number in parentheses.

To perform an operation, one must enter an operand and then either presses the desired operator button seen on the application or the operator key on one's keyboard. After all operands and operators have been appropriately placed the user can now press either the enter key on their keyboard or the "=" button on the application to solve the expression.

To close the application, press the X button in the upper right hand corner.

To change the language, select the Setting menu, go to Languages, and select the desired language.

If a mistake was made when entering the current operand, the clear button can be used to erase the entire current operand. The user can also use the backspace button or backspace key to delete the last character of the current opperand.

Uniary Operators

In order to use uniary operators, such as inverse, exponents, and square root, the user must first type the operand and then click on the desired uniary operator. Once the desired operator appears on the screen, choose a binary operator to proceed to the next operand that will be entered, or press the "=" button or enter key to solve the expression.

History

The history will stored any and all calculations that have been solved during a session. To open the history window, one must press the ">" button that can be seen on the right hand side of the application. To close the history the user must press the "<" button seen on the right hanf side of the history display.

Downloading

If the user wants to download the calculations in the history display, the user needs to go to the File menu and select Download. There will be a File dialog that will show and the user will have to select the desired location for the file.

Printing

If the user wants to print the calculations in the history display, the user needs to go to the File menu and select Print. From there a print dialog will show and the user will have to select the desired printer that will print the calculations.


Información General

Rimplex fue diseñado para trabajar con números complejos y realizar operaciones simples en ellos. Para colocar un número complejo como un solo operando en una expresión, uno debe colocar el número complejo entre paréntesis.

Para realizar una operación, se debe ingresar un operando y luego presionar el botón del operador deseado que se ve en la aplicación o la tecla del operador en el teclado. Después de que todos los operandos y operadores se hayan colocado correctamente, el usuario ahora puede presionar ya sea la tecla Intro en su teclado o el botón "=" en la aplicación para resolver la expresión.

Para cerrar la aplicación, presione el botón X en la esquina superior derecha.

Si se cometió un error al ingresar el operando actual, el botón borrar se puede usar para borrar todo el operando actual. El usuario también puede usar el botón de retroceso o la tecla de retroceso para eliminar el último carácter del operador actual.

Operadores uniares

Para utilizar operadores uniares, como inversos, exponentes y raíz cuadrada, el usuario primero debe escribir el operando y luego hacer clic en el operador uniario deseado. Una vez que el operador deseado aparezca en la pantalla, elija un operador binario para pasar al siguiente operando que será ingresado, o presione el botón "=" o la tecla enter para resolver la expresión.

Historial

El historial almacenará todos y cada uno de los cálculos que se hayan resuelto durante una sesión. Para abrir la ventana del historial, se debe presionar el botón ">" que se puede ver en el lado derecho de la aplicación. Para cerrar el historial, el usuario debe presionar el botón "<" que se ve en el lado derecho de la pantalla del historial.

Descargando

Si el usuario desea descargar los cálculos en la pantalla del historial, debe ir al menú Archivo y seleccionar Descargar. Habrá un cuadro de diálogo Archivo que se mostrará y el usuario tendrá que seleccionar la ubicación deseada para el archivo.

Impresión

Si el usuario desea imprimir los cálculos en la pantalla del historial, debe ir al menú Archivo y seleccionar Imprimir. Desde allí, se mostrará un cuadro de diálogo de impresión y el usuario tendrá que seleccionar la impresora deseada que imprimirá los cálculos.


Informations Générales

Rimplex a été conçu pour travailler avec des nombres complexes et effectuer des opérations simples sur eux. Afin de placer un nombre complexe en tant qu'opérande unique dans une expression, il faut placer le nombre complexe entre parenthèses.

Pour effectuer une opération, il faut saisir un opérande puis soit appuyer sur le bouton opérateur souhaité vu sur le l'application ou la touche opérateur de son clavier. Une fois que tous les opérandes et opérateurs ont été correctement placés, l'utilisateur peut maintenant appuyer sur soit la touche Entrée de leur clavier, soit le bouton "=" de l'application pour résoudre l'expression.

Pour fermer l'application, appuyez sur le bouton X dans le coin supérieur droit.

Pour modifier la langue, sélectionnez le menu Paramètres, accédez à Langues et sélectionnez la langue souhaitée.

Si une erreur a été commise lors de la saisie de l'opérande actuel, le bouton d'effacement peut être utilisé pour effacer tout l'opérande actuel. L'utilisateur peut également utiliser le bouton de retour arrière ou la touche de retour arrière pour supprimer le dernier caractère de l'opperande en cours.

Opérateurs unitaires

Pour utiliser des opérateurs uniaires, tels que l'inverse, les exposants et la racine carrée, l'utilisateur doit d'abord taper l'opérande puis cliquer sur l'opérateur unitaire souhaité. Une fois que l'opérateur souhaité apparaît à l'écran, choisissez un opérateur binaire pour passer à l'opérande suivant qui sera entré, ou appuyez sur le bouton "=" ou entrez la touche pour résoudre l'expression.

Historique

L'historique stockera tous les calculs qui ont été résolus au cours d'une session. Pour ouvrir la fenêtre d'historique, il faut appuyer sur le bouton ">" visible sur le côté droit de L'application. Pour fermer l'historique, l'utilisateur doit appuyer sur le bouton "<" qui se trouve sur le côté droit de l'affichage de l'historique.

Téléchargement

Si l'utilisateur souhaite télécharger les calculs dans l'affichage de l'historique, il doit aller dans le menu Fichier et sélectionner Télécharger. Une boîte de dialogue Fichier s'affichera et l'utilisateur devra sélectionner l'emplacement souhaité pour le fichier.

Impression

Si l'utilisateur souhaite imprimer les calculs dans l'affichage de l'historique, il doit aller dans le menu Fichier et sélectionner Imprimer. De là, une boîte de dialogue d'impression apparaîtra et l'utilisateur devra sélectionner l'imprimante souhaitée qui imprimera les calculs.
